# Django Master by PyCodeBR

## O que é Django e para o que ele serve?

### O que é?

- Framework web
- Para Python, feito com Python
- Código aberto(OpenSource)
- Arquitetura MVT(Model View Template)
- Fullstack

### Para que serve?

- Sites, blogs, redes sociais
- Sistemas web completos
- ERPs
- CRMs
- PDVs
- APIs(DRF)
